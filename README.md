# Rule-Based Impulse Spending Detection Using Behavioral and Transaction Data with Adaptive Budgeting System

## Author
Danyah Sohail Abid  
FAST National University of Computing and Emerging Sciences  
Islamabad, Pakistan  

## Abstract
This project presents a lightweight rule-based framework for detecting impulsive spending behavior using transaction and behavioral data. The system integrates adaptive budgeting mechanisms to provide real-time financial guidance without relying on computationally expensive machine learning models.

## Features
- Real-time impulse spending detection
- Adaptive budgeting system
- Rule-based lightweight architecture
- Behavioral and transaction analysis
- Financial decision support

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
The dataset was obtained from Kaggle and includes:
- Transaction Amount
- Category
- Transaction Time
- Transaction Type

## Results
- Accuracy: 82%
- Precision: 79%
- Recall: 76%
- F1-Score: 77%
